Этот репозиторий представляет собой портфолио моих работ и учебных проектов в различных IT-направлениях.

Проекты сгруппированы по областям:

Python — прикладные задачи и проекты, написанные на Python

DO (DevOps) — проекты, связанные с автоматизацией, CI/CD, Docker и системным администрированием

SQL — проекты по работе с PostgreSQL, проектированию схем и написанию запросов

BSA (Business & System Analysis) — проекты по бизнес- и системной аналитике, включая документацию и моделирование систем

Каждая директория содержит собственный README с подробным описанием контекста, требований и реализованных решений.
